# 珠海剥皮人 (Zhuhai Skinner)

1990 年代珠海街道背景的 2D 复古恐怖逃脱游戏。纯 HTML5 Canvas + 原生 ES Modules JavaScript，零依赖、零构建，全部美术与音频均为程序化生成。

## 运行
直接用 HTTP 服务器托管本目录即可（ES Modules 需要 http 协议）：

```bash
python3 -m http.server 8080
# 打开 http://localhost:8080
```

## 玩法
你是蒋阳宏。1994 年深夜的珠海小区，剥皮人刘铮铮正在游荡。找到 **3 把大门钥匙**，从小区东侧铁门逃出。

- 翻垃圾桶、拆纸箱获取道具与钥匙
- 木柜、灌木丛可以躲藏；房门可以开关（关门能甩掉追击）
- 电话亭存档；井盖随机传送；木板筑障碍；鞭炮引开敌人
- 手电筒照亮前路，但也会让剥皮人更容易发现你

## 操作
| 按键 | 功能 |
|---|---|
| WASD / 方向键 | 移动 |
| Shift | 奔跑（消耗体力） |
| F | 手电筒 |
| E | 交互（搜刮/躲藏/开关门/存档/传送） |
| Q | 使用道具 |
| 1-4 | 选择背包格 |
| Esc | 暂停 |
